* Usage

+ =hddtemp= :: Show temps for all =sd= and =nvme= disks.
+ =hddtemp /dev/sda ...= :: Show temps for selected disks

Options:

- =-h, --help= :: Show usage
- =-q= :: Suppress warnings
- =-V, --version= :: Show version

* Requirements

- =smartctl= (=smartmontools= package)
  - NVMe support: =smartctl= v6.5+
